Documentação da API

Bem-vindo a documentação da API de livros.

Esta é a API desenvolvida para fins de estudo e apresentação do desafio final do curso FullStack proporcionado pelo Vai na Web.

Aqui são utilizados as tecnologias de API Flask e SQLite.

A API, neste momento, possui três endpoints, os quais: permitem visualizar esta documentação, o cadastro e a listagem de livros doados.

Endpoints

Abaixo a listagem de endpoints, clique em cada um para ver os detalhes:

Página Inicial (GET /)

Retorna esta documentação.

Resposta de Sucesso (200):

  • Está página mesmo :D
Doar um livro (POST /api/doar)

Adiciona um novo livro ao banco de dados.

Corpo da Requisição (JSON):

  • "title": "Nome do Livro"
  • "category": "Categoria"
  • "author": "Autor"
  • "image_url": "URL da Imagem"

Resposta de Sucesso (201):

  • {
  • message": "Livro cadastrado com sucesso"
  • livro": {
  • "id": 1,
  • "title": "Nome do Livro",
  • "category": "Categoria",
  • "author": "Autor",
  • "image_url": "URL da Imagem"
  • }
Listar todos os livros (GET /api/livros)

Retorna uma lista de todos os livros cadastrados.

Resposta de Sucesso (200):

  • [
  • {
  • "id": 1,
  • "title": "Nome do Livro",
  • "category": "Categoria",
  • "author": "Autor",
  • "image_url": "URL da Imagem"
  • }
  • ]
